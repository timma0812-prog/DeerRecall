# Homepage Finalized Reference

This folder is the frozen visual reference for the DeerRecall homepage before formal client development.

## States

- `empty-state.html`: default homepage search state.
- `results-state.html`: homepage after a DeerSearch query returns candidates.
- `homepage-reference.css`: shared visual system for both states.

## Screenshots

- `screenshots/empty-state.png`
- `screenshots/results-state.png`

Use these files as the source of truth for layout proportions, color, spacing, card density, and right-panel hierarchy when implementing the production client.
